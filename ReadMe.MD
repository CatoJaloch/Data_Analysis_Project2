# create a virtual environment

**Windows**
`python -m venv venv`

**Ubuntu**
`python3 -m venv venv`

# activate virtual environment

**Windows**
`venv\Scripts\activate`

**Ubuntu**
`source venv/bin/activate`

# install dependencies

**Windows & Ubuntu**
`pip install -r requirements.txt`

# run script

`python -m src.monitor.performance`
